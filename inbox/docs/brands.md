# Производители подшипников

[🏠 На главную](../../../README.md) | [📖 Справочник подбора](../selection_guide.md) | [👨‍💼 Для менеджеров](../training/managers_guide.md)

_Дата обновления: 2025-12-28_

## Purpose
- Зафиксировать перечень производителей подшипников и сопутствующих изделий
- Использовать данные для анализа ассортимента и проверки наличия брендов в закупочных спецификациях

## Classification
- Категория: производители/бренды подшипников, уплотнений, приводных ремней и компонентов линейного перемещения
- Область применения: справочные материалы по рынку подшипников и совместимых комплектующих

## Designation
- **Бренд** — коммерческое обозначение производителя из каталога
- **Страна** — страна происхождения бренда; пустые значения помечены как «не указано»
- **Специализация** — краткое описание продукции из каталога

## Prefixes/Suffixes
- Для наименований брендов префиксы/суффиксы не используются

## Examples
- `SKF` (Швеция) — подшипники
- `HABASIT` (Швейцария) — конвейерные ленты и приводные ремни
- `ARTNC` (страна не указана) — подшипники, рейки, шестерни, винты, гайки, муфты

## Notes & Limitations
- Данные приведены в порядке публикации без корректировки формулировок
- Позиции без явного указания страны помечены как «не указано»
- При обновлении страницы источник может изменить порядок или перечень брендов; обновлять выгрузку при изменениях.

## Standard References
- [/brands.php](/brands.php) — официальный список производителей на момент выгрузки.
- Локальный источник: `sources/brands.json` (JSON-выгрузка 284 записей).
