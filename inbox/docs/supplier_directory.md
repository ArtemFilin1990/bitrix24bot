# 📚 Каталог производителей и брендов подшипников

[🏠 На главную](../../../README.md) | [📖 Справочник подбора](../selection_guide.md) | [👨‍💼 Для менеджеров](../training/managers_guide.md) | [🌍 Международные бренды](international_brands.md)

---

## Purpose / Назначение
- Систематизированный перечень производителей, брендов и торговых марок подшипников и сопутствующей продукции.
- Быстрый справочник для менеджеров и инженеров: проверка, поставляется ли бренд и в какой товарной категории он работает.
- Отдельный реестр поставщиков без смешения со стандартами ГОСТ/ISO и таблицами аналогов.

## Classification / Классификация
- Бренды приведены в алфавитном порядке без оценок качества или цены (см. `international_brands.md` для сегментации).
- Основные товарные направления сведены в колонке «Категория»: подшипники, ремни, узлы линейного перемещения, уплотнения и др.
- Страна указана по данным поставщика; комбинированные записи сохраняют исходное написание (например, «Япония-Франция»).

## Designation / Формат записи
- Каждая строка таблицы описывает **бренд**, его **профиль** (официальное позиционирование), **страну производства/регистрации** и **категорию поставки**.
- Если страна в исходных данных отсутствует, в таблице указано «— (не указано)» — требуется уточнение по каталогу производителя.
- Названия брендов сохранены в оригинальном регистре; кириллические наименования переданы без транслитерации.

## Prefixes/Suffixes / Префиксы и суффиксы
- Не используются: бренды приведены в официальном написании из списка поставщика без модификаций.
- Альтернативные обозначения (например, написание через пробел или дефис) даны так, как они присутствуют в источнике.

## Examples / Примеры разбора
- `SKF | Подшипники | Швеция | Подшипники` → шведский бренд, основной ассортимент — подшипники качения.
- `ARTNC | Подшипники, рейки, шестерни, винты, гайки, муфты | — (не указано) | Подшипники, рейки, шестерни, винты, гайки, муфты` → страна не указана в исходном списке, требуется верификация по каталогу.

## Notes & Limitations / Ограничения
- Данные приведены в исходном виде, без нормализации ассортиментных линеек и без проверки наличия официальных представительств.
- Для брендов с пометкой «— (не указано)» нужно дополнить страну по официальным каталогам перед коммерческим использованием.
- Перечень не заменяет стандарты и не подтверждает взаимозаменяемость продукции — только каталог поставляемых марок.

## Standard References / Источники
- Перечень брендов от поставщиков (переданные входные данные). Требуется дополнительная сверка по официальным каталогам производителей.
