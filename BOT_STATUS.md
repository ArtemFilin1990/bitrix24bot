## 🚀 Как развернуть бота

### Вариант 1: Через GitHub Actions (рекомендуется)

1. **Настройте секреты GitHub** (один раз):
   - `CLOUDFLARE_API_TOKEN`
   - `CLOUDFLARE_ACCOUNT_ID`

2. **Установите секреты через Wrangler**:
   ```bash
   wrangler secret put GEMINI_API_KEY
   wrangler secret put B24_PORTAL           # ewerest.bitrix24.ru
   wrangler secret put B24_USER_ID          # 1
   wrangler secret put B24_TOKEN            # p7mpsrj88h2ustgk
   wrangler secret put B24_APP_TOKEN        # z9rxpcoeaslfm10j04vybah79h2jumma
   wrangler secret put WORKER_HOST          # bitrix24bot.vza3555.workers.dev
   wrangler secret put IMPORT_SECRET        # любая случайная строка
   ```

3. **Запустите деплой**:
   ```bash
   # Автоматически при push в main
   git push origin main

   # Или вручную через GitHub Actions
   Actions → Deploy to Cloudflare Workers → Run workflow
   ```

4. **Загрузите данные** (один раз):
   ```bash
   Actions → Seed Database → Run workflow
   ```

### Вариант 2: Локально

```bash
# 1. Установите wrangler
npm install -g wrangler@4.76.0

# 2. Авторизуйтесь
wrangler login

# 3. Разверните
npm run deploy

# 4. Зарегистрируйте бота (один раз)
curl "https://bitrix24bot.vza3555.workers.dev/register?secret=<IMPORT_SECRET>"
```
